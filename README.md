# WarmTouch E-Commerce Store

**Domain:** [https://www.warmtotuch.store](https://www.warmtotuch.store)

## Overview
WarmTouch is a modern e-commerce platform specializing in handmade macrame, Ramadan decorations, coffee mugs, and home decor. The application is built with React for the frontend and features a robust backend for order management and authentication.

## Features
- **Responsive Design**: Optimized for mobile and desktop viewing.
- **Product Catalog**: Browse a wide range of categories including Macrame, Ramadan, Mugs, and Home Decor.
- **User Authentication**: Secure login and registration with email verification and Google OAuth.
- **Shopping Cart**: Fully functional cart and checkout process.
- **Order Tracking**: Detailed "My Orders" page with timeline status tracking.

## Tech Stack
- **Frontend**: React, React Router, Tailwind CSS, Lucide React
- **Backend**: Node.js, Express (Admin Dashboard)
- **Database**: MySQL (Aiven)

## Development

### Installation
1.  Clone the repository:
    ```bash
    git clone https://github.com/alfar-programer/Store-B.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

### Run Locally
Start the development server:
```bash
npm run dev
```
